# 🖥️ Amazon Egypt Gaming Laptop Scraper

This project is a web scraping and data processing pipeline built using Python and Selenium to extract gaming laptop data from Amazon Egypt and transform it into a structured dataset.

---

##  Project Overview

This project aims to:
- Scrape gaming laptop listings from Amazon Egypt
- Extract product information (title, price, link)
- Parse titles into structured specifications
- Build a clean dataset for analysis

---

##  Features

- Scrapes multiple pages
- Extracts:
  - Title
  - Price
  - Link

- Parses:
  - Brand
  - Series
  - CPU
  - RAM
  - Storage
  - GPU
  - Screen size
  - Refresh rate
  - Operating system
  - Color

---

##  Technologies Used

- Python
- Selenium
- Pandas
- Regular Expressions (re)
